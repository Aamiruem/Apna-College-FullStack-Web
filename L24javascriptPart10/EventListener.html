JavaScript Event Listeners

Event listeners are functions that wait for a specific event to occur on an element, then execute code in response.

1. Click Event

The most common event listener - responds to mouse clicks.

Click Me Too!
Waiting for click event...
// JavaScript code for click event
document.getElementById('clickButton').addEventListener('click', function() {
  console.log('Button was clicked!');
  clickOutput.textContent = 'Button was clicked at: ' + new Date().toLocaleTimeString();
});

2. Double-Click Event

Responds to double-click actions.

Double Click Me!
Waiting for double-click event...
// JavaScript code for double-click event
doubleClickBox.addEventListener('dblclick', function() {
  this.style.transform = 'rotate(45deg)';
  doubleClickOutput.textContent = 'Double-click detected!';
});

3. Mouse Events

Events related to mouse movements and actions.

Hover Over Me!
Mouse events will appear here...
// JavaScript code for mouse events
mouseBox.addEventListener('mouseenter', function() {
  mouseOutput.textContent = 'Mouse entered the box';
});

mouseBox.addEventListener('mouseleave', function() {
  mouseOutput.textContent = 'Mouse left the box';
});

4. Keyboard Events

Events triggered by keyboard actions.

Keyboard events will appear here...
// JavaScript code for keyboard events
keyInput.addEventListener('keydown', function(event) {
  keyOutput.textContent = `Key pressed: ${event.key} (Key code: ${event.code})`;
});

5. Event Object Properties

Every event listener receives an event object with useful properties.

Click Anywhere
Event details will appear here...
// JavaScript code for event object
document.addEventListener('click', function(event) {
  eventObjectOutput.innerHTML = `Clicked at X:${event.clientX}, Y:${event.clientY}
  Target: ${event.target.tagName}`;
});
