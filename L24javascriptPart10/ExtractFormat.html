Extracting Form Data with JavaScript

Extracting Form Data with JavaScript

Learn different methods to extract data from HTML forms using JavaScript

User Registration Form

Method 1: FormData Object

The FormData object provides a way to easily construct a set of key/value pairs representing form fields and their values.

// Using FormData to extract form values const form = document.getElementById('userForm'); const formData = new FormData(form); // Access individual values const username = formData.get('username'); const email = formData.get('email'); // Iterate through all values for (let [key, value] of formData.entries()) { console.log(key, value); }

Method 2: Direct Element Access

You can access form values directly through DOM elements using their ID or name attributes.

// Accessing form values directly const username = document.getElementById('username').value; const email = document.getElementById('email').value; const country = document.getElementById('country').value; // For checkboxes and radio buttons const newsletter = document.getElementById('newsletter').checked; const gender = document.querySelector('input[name="gender"]:checked')?.value;

Method 3: Form Element Access

Form elements can be accessed through the form object itself using their name attributes.

// Accessing form values through form element const form = document.getElementById('userForm'); const username = form.username.value; const email = form.email.value; const country = form.country.value; // For checkboxes const newsletter = form.newsletter.checked;
