What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content.

The DOM represents a document as a tree of nodes. The nodes are objects that can be manipulated programmatically with JavaScript.

Selecting DOM Elements

JavaScript provides several methods to select elements from the DOM:

// Select by ID
const header = document.getElementById('header');

// Select by class name
const items = document.getElementsByClassName('item');

// Select by tag name
const paragraphs = document.getElementsByTagName('p');

// CSS selector (returns first match)
const element = document.querySelector('.container');

// CSS selector (returns all matches)
const elements = document.querySelectorAll('.item');

Manipulating Elements

Once you've selected elements, you can manipulate their content and attributes:

// Change text content
element.textContent = 'New text';

// Change HTML content
element.innerHTML = '<strong>Bold text</strong>';

// Change attributes
imageElement.src = 'new-image.jpg';

// Change styles
element.style.color = 'blue';
element.style.fontSize = '20px';

// Add/remove CSS classes
element.classList.add('highlight');
element.classList.remove('old-class');

Creating and Removing Elements

You can dynamically create new elements and add them to the DOM, or remove existing ones:

// Create new element
const newElement = document.createElement('div');
newElement.textContent = 'I am a new div!';

// Add to the DOM
parentElement.appendChild(newElement);

// Insert at specific position
parentElement.insertBefore(newElement, referenceElement);

// Remove element
parentElement.removeChild(elementToRemove);

// Replace element
parentElement.replaceChild(newElement, oldElement);

Event Handling

JavaScript allows you to handle user interactions through events:

// Inline event handling
<button onclick="handleClick()">Click me</button>

// Using event listeners
const button = document.querySelector('button');
button.addEventListener('click', function() {
    console.log('Button clicked!');
});

// Common events: click, mouseover, mouseout, keydown, keyup, change, submit

DOM Traversal

Once you have a reference to an element, you can navigate to related elements:

// Get parent element
const parent = element.parentNode;

// Get children
const children = element.children;
const firstChild = element.firstElementChild;
const lastChild = element.lastElementChild;

// Get siblings
const nextSibling = element.nextElementSibling;
const previousSibling = element.previousElementSibling;
