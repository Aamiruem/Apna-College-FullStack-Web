Event Object Properties

When a mouse event occurs, JavaScript passes an event object to the handler function with useful properties:

clientX / clientY

Mouse coordinates relative to the viewport

pageX / pageY

Mouse coordinates relative to the document

offsetX / offsetY

Mouse coordinates relative to the target element

button

Which mouse button was pressed (0: left, 1: middle, 2: right)

buttons

Which mouse buttons are currently pressed

altKey / ctrlKey / shiftKey

Whether modifier keys were pressed during the event
