// Add to the container

document.createElement()

The document.createElement(tagName) method creates a new HTML element specified by the tag name. The element is created in memory but not yet added to the document.

After creation, you can modify the element's properties, styles, and content before adding it to the DOM.

DOM Insertion Methods

appendChild(): Adds a node to the end of the list of children of a specified parent node.

append(): Inserts a set of Node objects or DOMString objects after the last child of the parent.

prepend(): Inserts a set of Node objects or DOMString objects before the first child of the parent.

insertAdjacentElement(): Inserts a given element node at a given position relative to the element it is invoked upon.
