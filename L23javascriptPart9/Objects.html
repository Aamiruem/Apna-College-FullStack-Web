Setting Content in JavaScript Objects

1. Dot Notation

Setting properties using dot notation

const obj = {};
obj.name = "John";
obj.age = 30;

2. Bracket Notation

Setting properties using bracket notation

const obj = {};
obj["first name"] = "Jane"; // Works with spaces
obj["age"] = 25;

3. Computed Property Names

Using expressions as property names (ES6 feature)

const dynamicKey = "score";
const obj = {
    name: "Alex",
    [dynamicKey]: 95,  // Correct usage
    ["grade" + dynamicKey.slice(5)]: "A" // Another example
};

4. Object.defineProperty

Defining properties with specific attributes

const obj = {};
Object.defineProperty(obj, 'readOnlyProp', {
    value: "Can't change me",
    writable: false,
    enumerable: true
});

5. Object.assign

Copying properties from one or more source objects

const obj1 = { a: 1, b: 2 };
const obj2 = { b: 3, c: 4 };
const result = Object.assign({}, obj1, obj2);

6. Spread Operator (ES6)

Merging objects using the spread operator

const obj1 = { a: 1, b: 2 };
const obj2 = { b: 3, c: 4 };
const result = { ...obj1, ...obj2 };
